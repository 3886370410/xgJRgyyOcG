# 课堂骰票系统简介 python1154

## 项目概述

课堂骰票系统是一个基于Python语言和Django框架开发的B/S架构应用程序，旨在提供一种方便、安全的在线投票解决方案，主要服务于学生和管理员两类用户。

## 技术栈

- **开发语言**: Python
- **架构**: B/S架构
- **数据库**: MySQL
- **框架**: Django

## 功能模块

### 学生模块

- **个人信息注册**：学生可注册个人信息，并通过学号或身份证号码等正规身份验证机制，保障用户身份的准确性和安全性。
- **投票参与**：注册学生可以使用系统进行投票。
- **历史投票查询**：学生可查询个人历史投票记录及投票统计结果。

### 管理员模块

- **投票管理**：管理员可进行投票的发布、修改、删除等操作，并可自由设置投票内容、选项、时间和规则等。
- **投票分析**：管理员可以查看投票分析结果，系统提供及时准确的数据统计和分析，并支持数据可视化功能。

## 系统角色
